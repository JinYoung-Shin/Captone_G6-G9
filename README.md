# 인간친화 협업로봇
영상인식 라이브러리 OpenCV와 음성인식 라이브러리 VoiceRecognition(예정)을 활용하여 볼트 너트를 인식하고 조이는 작업까지 진행하는 로봇 팔 입니다.

## 1.사용언어
- Python 3.xx

## 2.사용한 라이브러리
- Google Voice Recognition
- Intel OpenCV
- Dobot API
